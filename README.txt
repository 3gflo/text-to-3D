Text-to-3D Project Setup

1. Install packages:
   pip install -r requirements.txt

2. Create .env file with:
   GEMINI_API_KEY=your_key
   HF_TOKEN=your_token

3. Get API keys:
   - Gemini: https://ai.google.dev/
   - HuggingFace: https://huggingface.co/settings/tokens

4. Run:
   python main.py
